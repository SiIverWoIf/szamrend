PowerShell Parancsok - Jegyzet

Alapok

Alapvető parancsok és szintaxis.

Szöveg kiírása

echo "szöveg": Kiírja a megadott szöveget a konzolra.


echo "Hello World"  # Kimenet: Hello World

Változók

$változóNév = érték: Változó deklarálása és értékadás.


$myVar = 10
echo $myVar  # Kimenet: 10

Változó növelése

$változóNév++: Növeli a változó értékét 1-gyel.


$myVar = 5
$myVar++
echo $myVar  # Kimenet: 6

Megjegyzés

#: Egysoros megjegyzés. A # jel utáni rész nem kerül végrehajtásra.


# Ez egy megjegyzés

Argumentumok kezelése

$args: A szkriptnek átadott parancssori argumentumokat tartalmazó tömb.


# args.ps1 10 20
echo $args[0]  # Kimenet: 10
echo $args[1]  # Kimenet: 20
echo $args.Count  # Kimenet: 2

Feltételes elágazások (if-else)

Feltételek alapján történő vezérlés.

  • if (feltétel) { parancsok }: Ha a feltétel igaz, a kapcsos zárójelek közötti parancsok végrehajtásra kerülnek.
  • elseif (feltétel) { parancsok }: További feltételek megadására szolgál.
  • else { parancsok }: Ha egyik feltétel sem igaz, az else ágban lévő parancsok hajtódnak végre.

Összehasonlító operátorok:

  • -eq: egyenlő
  • -ne: nem egyenlő
  • -gt: nagyobb, mint
  • -ge: nagyobb vagy egyenlő
  • -lt: kisebb, mint
  • -le: kisebb vagy egyenlő

$num = 15

if ($num -gt 10) {
    echo "A szám nagyobb, mint 10"
} elseif ($num -eq 10) {
    echo "A szám egyenlő 10-zel"
} else {
    echo "A szám kisebb, mint 10"
}

Ciklusok (for)

for ($i = kezdőérték; $i -le végérték; $i++) { parancsok }: A ciklusváltozó (`$i`) a kezdőértéktől indul, a megadott lépésközzel (`$i++`) növekszik a végértékig, és minden ciklusban végrehajtja a kapcsos zárójelek közötti parancsokat.


for ($i = 0; $i -le 5; $i++) {
    echo "i = $i"
}
# Kimenet:
# i = 0
# i = 1
# i = 2
# i = 3
# i = 4
# i = 5

Objektumok

[PSCustomObject]@{ Tulajdonság1 = "Érték1"; Tulajdonság2 = "Érték2" }: Egyéni objektum létrehozása.


$person = [PSCustomObject]@{
    Name = "John Doe"
    Age = 30
}
echo $person.Name  # Kimenet: John Doe
echo $person.Age  # Kimenet: 30

Objektumok tömbje


$people = @(
    [PSCustomObject]@{ Name = "Alice"; Age = 25 }
    [PSCustomObject]@{ Name = "Bob"; Age = 35 }
)
echo $people[0].Name  # Kimenet: Alice

Parancsok objektumok kezelésére

Write-Host

Kiírja az objektum tulajdonságait formázva.


$person = [PSCustomObject]@{ Name = "John"; Age = 30 }
Write-Host "Name: $($person.Name), Age: $($person.Age)"
# Kimenet: Name: John, Age: 30

Sort-Object

Objektumok rendezése adott tulajdonság alapján. -Descending: Csökkenő sorrend.


$people | Sort-Object Age -Descending
# Az objektumokat az Age tulajdonság alapján rendezi csökkenő sorrendben.

Select-Object

Objektumok bizonyos tulajdonságainak kiválasztása.


$people | Select-Object Name, Age
# Csak a Name és Age tulajdonságokat adja vissza.

Measure-Object

Objektumok számlálása, összegzése, átlagolása. -Line: Sorok számlálása. -Count: Elemek számlálása. -ExpandProperty: Tulajdonság értékének lekérése. 


# A numbers.txt fájl sorainak száma:
Get-Content .\numbers.txt | Measure-Object -Line | Select-Object -ExpandProperty Lines

Fájlkezelés

Get-Content

Fájl tartalmának beolvasása.


$fileContent = Get-Content "myFile.txt"
echo $fileContent # Kiírja a fájl tartalmát

Test-Path

Fájl vagy mappa létezésének ellenőrzése. -PathType Leaf: Fájl létezését ellenőrzi.


if (Test-Path "myFile.txt" -PathType Leaf) {
    echo "A fájl létezik"
}

Stringkezelés

.Split("elválasztó"): Szöveg feldarabolása egy elválasztó karakter mentén.


$line = "apple banana orange"
$words = $line.Split(" ")
echo $words[1]  # Kimenet: banana

Függvények

function FüggvényNév { parancsok }: Függvény definiálása.

param([típus]$paraméternév): Paraméter definiálása a függvényen belül.

FüggvényNév -paraméternév érték: Függvényhívás paraméterátadással.


function Greet {
    param (
        [string]$name
    )
    echo "Hello, $name!"
}

Greet -name "Peter"  # Kimenet: Hello, Peter!

Matematikai műveletek

  • +, -, *, /: Összeadás, kivonás, szorzás, osztás.
  • [math]::Sqrt(szám): Négyzetgyökvonás.

$sum = 10 + 5  # Kimenet: 15
$diff = 10 - 5 # Kimenet: 5
$prod = 10 * 5 # Kimenet: 50
$quot = 10 / 5 # Kimenet: 2
$sqrt = [math]::Sqrt(9)  # Kimenet: 3

Másodfokú egyenlet megoldása


$a = [double](Read-Host "Enter the coefficient 'a'")
$b = [double](Read-Host "Enter the coefficient 'b'")
$c = [double](Read-Host "Enter the coefficient 'c'")

$discriminant = $b * $b - 4 * $a * $c

if ($discriminant -gt 0) {
    $x1 = (-$b + [math]::sqrt($discriminant)) / (2 * $a)
    $x2 = (-$b - [math]::sqrt($discriminant)) / (2 * $a)
    echo $x1
    echo $x2
}
elseif ($discriminant -eq 0) {
    $x1 = -$b / (2 * $a)
    echo $x1
}
else {
    echo "Nincs valos megoldas"
}

Prímteszt függvény


function Test-Prime {
    param ( [int]$number )

    if ($number -le 1) { return $false } # 0, 1 nem prímek
    if ($number -le 3) { return $true }  # 2 és 3 prímek

    if ($number % 2 -eq 0 -or $number % 3 -eq 0) { return $false } # Osztható 2-vel vagy 3-mal

    for ($i = 5; $i * $i -le $number; $i += 6) { # Optimalizált ciklus
        if ($number % $i -eq 0 -or $number % ($i + 2) -eq 0) {
             return $false
        }
    }

    return $true
}

Test-Prime -number 23 # Kimenet: True
Test-Prime -number 24 # Kimenet: False

Tippek

  • Használd a Tab billentyűt a parancsok és változónevek automatikus kiegészítéséhez.
  • Használd a Get-Help parancsnév parancsot a súgó megtekintéséhez.
  • A PowerShell ISE (Integrated Scripting Environment) egy hasznos eszköz a szkriptek írásához és futtatásához. A Windows keresőjébe írva megtalálod.
  • A VS Code (Visual Studio Code) egy másik népszerű szerkesztő, amely támogatja a PowerShell szkriptek írását és futtatását, rengeteg bővítménnyel.
